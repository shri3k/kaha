About

Kaha.co as name suggests helps you find help and supplies. You can go through different categories on sidemenu and limit your results by location (districts and tole names).
If you are providing or know someone who is providing any help or supplies please use Add a supply/resource to add it.
Please be as descriptive as possible.

How to use Kaha

1. What does the color code mean?
Green color indicates avaialble resources/supplie; Red indicates resources needed. See contact details to find out how you can use the suplies or help needed ones.
2. How can I search items?
Navigate to the appropriate supplies and type in what you are looking for. Example: Type in 'supply' or need' or 'kathmandu' or 'government' etc.
3. How can I add new supply?
Navigate to the section where you want to add supply. Then click on "Plus button" at bottom and click 'I supply'.
4.How can I add new resources I need?
Navigate to the section from which you need supply. Then click on "Plus button" at bottom and click 'I need'.
5. How can I report that a supply is not valid?
Navigate to the appropriate supplies and in the detail description click 'No longer available'. On the right side is the number which indicates how many times the button is clicked. If the counter number is increasing we will remove the item.
6. How can I delete supply?
Navigate to the appropriate supplies and in the detail description click 'Submit a request to remove'. On the right side is the number which indicates how many times the button is clicked. If the counter number is increasing we will delete the item.
7. How can I update/edit item?
Navigate to the appropriate supplies and in the detail description click 'Edit Item' on top right corner. Make change to the item you want and click submit button. The item will be saved.
